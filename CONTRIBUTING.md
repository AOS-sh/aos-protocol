# Contributing to AOS Protocol

Love it? Help make it the standard.

## Ideas
- Open an issue: "New example for [WordPress/ecommerce/blog]"
- Suggest v0.2 features: rate limits? Signatures?

## Code
- Fork → Add example/template → PR
- Keep it simple: No breaking changes to v0.1

## Ship & Share
- Add .agent to your site
- Tweet `@sir__fox #aosprotocol` with your URL
- We'll add it to the leaderboard

Everyone's welcome. Let's own the agent era.
